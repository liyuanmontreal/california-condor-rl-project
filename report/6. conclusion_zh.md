# 总结

本项目以加州神鹰为案例，展示了强化学习在濒危物种管理中的可行性。

主要成果包括：

1. 构建了经校准的年龄结构生态模型  
2. 成功再现历史种群变化趋势  
3. 使用 Fitted Q-Iteration 训练离线 RL 管理策略  
4. 通过多种图表（轨迹、相图、KDE、hexbin、热力图）可视化策略结构  
5. 从生态学角度解释 RL 策略行为  

研究表明：

- 在强生态动力学背景下，RL 倾向学习节约成本的策略  
- 种群在随机灾害之外保持长期稳定  
- 策略呈现多模态、弱状态依赖结构  
- RL 生成的管理建议具有生物学合理性与稳健性  

本研究为未来基于 RL 的保护管理工具奠定了基础。
